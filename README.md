# 🎮 Full Stack Tic Tac Toe

A modern implementation of the classic Tic Tac Toe game.
Built to demonstrate client-server communication using React and Express.

## 🚀 Features
* **Architecture:** Monorepo with separated Client (React) and Server (Node.js).
* **Tech Stack:** TypeScript, Material UI (MUI), Express.js.
* **Functionality:** Tracks current session score vs. global server-wide score using real-time polling.

## 📦 How to Run locally
1.  Clone the repo.
2.  **Server:** `cd server` -> `npm install` -> `npm run dev`
3.  **Client:** `cd client` -> `npm install` -> `npm start`
